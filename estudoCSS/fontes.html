Fontes

Declarações

  • font-style
    • normal
    • italic
  • font-weight
    • Define o "peso", que é basicamente o quão negrito vai ficar:
    • 100 - 900
    • lighter
    • normal
    • bold
    • bolder
  • font-size
    • Medidas absolutas: cm, mm, in, px, pt, pc.
    • Medidas relativas: em, ex, rem, vw, vh, %.
    • Recomendado:px / em.
  • font-family
    • Use no maximo 4 fontes no site.
    • Fontes do google
      •
    • Baixando fontes externas, você deve colocar na pasta do projeto e utilizar o seguinte codigo para criar uma ''font-family'' com ela:
      • @font-face {
      src: url('nome da fonte.otf') format('opentype'), url('nome da fonte.ttf') format(truetype);
      Caso queira adicionar parametros adicionais para fonte:
      font-weight: normal; font-style: normal;
      }
      E ai para adicionar a fonte é só usar normalmente em font-family.

Pode usar uma declaração para unir todas essas 4 anteriores em uma só, que seria dessa forma:

font: italic bolder 3em 'Work sans', 'sans-serif';

Na ordem respectiva

  1. font-style
  2. font-weight
  3. font-size
  4. font-family
